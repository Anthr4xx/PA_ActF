			==========================================
			|       Dependencias con pipenv          |
			==========================================

1. En la terminal:
   cd /ruta/al/proyecto

   Ejemplo:
   cd D:\Users\Andres\Desktop\Programación Avanzada\Actividad Final

2. Instalar dependencias con Pipfile.lock:
   pipenv install

3. Activar el entorno virtual:
   pipenv shell



			==========================================
			|        Ejecutar sin setup.py           |
			==========================================
1. Dentro del cmd, en la raíz del proyecto. Ir a la carpeta shmup:
   cd shmup

2. Correr el juego directamente:
   python game.py



			==========================================
			|        Ejecutar con setup.py           |
			==========================================
1. Dentro del cmd, en la raíz del proyecto., instalar el paquete:
   pip install .

2. Ejecutar el juego usando el comando:
   shmup
